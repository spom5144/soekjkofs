# Net Monitor Pro v1.0
## เครื่องมือตรวจสอบและวิเคราะห์เครือข่าย

### ผู้พัฒนา: ป้อม (Pom)
### ภาษาที่ใช้พัฒนา: C# (.NET 8.0 - Windows Forms)

---

## วิธีดาวน์โหลดและติดตั้ง

1. ดาวน์โหลดไฟล์ `NetMonitorPro_v1.0.zip`
2. แตกไฟล์ ZIP
3. ดับเบิลคลิกที่ `NetMonitorPro.exe` เพื่อเปิดโปรแกรม
4. **ไม่ต้องติดตั้ง .NET Runtime** เพิ่มเติม (โปรแกรมรวมมาในตัวแล้ว)

> **หมายเหตุ**: Windows อาจแสดงคำเตือน SmartScreen ให้คลิก "More info" แล้วเลือก "Run anyway"

---

## วิธีใช้งาน (คู่มือภาษาไทย)

### แท็บที่ 1: 🔍 สแกนเครือข่าย (Network Scanner)

ใช้สำหรับค้นหาอุปกรณ์ที่เชื่อมต่ออยู่ในเครือข่ายเดียวกัน

**วิธีใช้:**
1. เลือก **Network Interface** ที่ต้องการจากดรอปดาวน์ด้านบน
2. ระบุ **ช่วง IP** ที่ต้องการสแกน (เช่น `192.168.1`)
3. กดปุ่ม **"▶ เริ่มสแกน"**
4. โปรแกรมจะสแกน IP ตั้งแต่ .1 ถึง .254
5. อุปกรณ์ที่พบจะแสดงในตาราง พร้อมข้อมูล:
   - **IP Address**: ที่อยู่ IP ของอุปกรณ์
   - **Hostname**: ชื่อโฮสต์
   - **MAC Address**: ที่อยู่ MAC
   - **สถานะ**: ออนไลน์/ออฟไลน์
   - **เวลาตอบ**: ระยะเวลาในการตอบกลับ (มิลลิวินาที)
   - **ระบบปฏิบัติการ**: คาดเดาจากค่า TTL
6. กดปุ่ม **"⏹ หยุด"** เพื่อหยุดการสแกน

---

### แท็บที่ 2: 📦 ดักจับแพ็กเก็ต (Packet Capture)

แสดงข้อมูลการจราจรของเครือข่ายแบบเรียลไทม์ (โหมดจำลอง)

**วิธีใช้:**
1. กดปุ่ม **"▶ เริ่มจับ"** เพื่อเริ่มดักจับแพ็กเก็ต
2. ใช้ **ตัวกรอง** เพื่อแสดงเฉพาะโปรโตคอลที่ต้องการ:
   - ทั้งหมด, TCP, UDP, ICMP, HTTP, DNS
3. คลิกที่แพ็กเก็ตเพื่อดูรายละเอียด:
   - ข้อมูลต้นทาง/ปลายทาง
   - โปรโตคอลและขนาด
   - Hex Dump
4. กดปุ่ม **"⏹ หยุด"** เพื่อหยุดการดักจับ
5. กดปุ่ม **"🗑 ล้าง"** เพื่อลบข้อมูลทั้งหมด

**สีของแพ็กเก็ต:**
- 🔵 **TCP** = สีน้ำเงิน
- 🟢 **HTTP** = สีเขียว
- 🟡 **DNS** = สีส้ม
- 🟣 **TLS** = สีม่วง
- 🔴 **ARP** = สีแดงอ่อน

---

### แท็บที่ 3: 🔓 สแกนพอร์ต (Port Scanner)

ใช้สแกนพอร์ตที่เปิดอยู่บนเครื่องเป้าหมาย

**วิธีใช้:**
1. ระบุ **เป้าหมาย** (IP หรือ hostname) เช่น `192.168.1.1`
2. ระบุ **ช่วงพอร์ต** เช่น `1-1024` หรือ `80-443`
3. กดปุ่ม **"▶ เริ่มสแกน"**
4. พอร์ตที่เปิดจะแสดงในตาราง พร้อมข้อมูล:
   - **พอร์ต**: หมายเลขพอร์ต
   - **สถานะ**: เปิด/ปิด
   - **บริการ**: ชื่อบริการที่ทำงานบนพอร์ตนั้น (เช่น HTTP, SSH, FTP)
   - **แบนเนอร์**: ข้อมูลเวอร์ชันของบริการ
   - **เวลาตอบ**: ระยะเวลาในการเชื่อมต่อ

**พอร์ตที่รู้จักทั่วไป:**
| พอร์ต | บริการ |
|-------|--------|
| 21 | FTP |
| 22 | SSH |
| 25 | SMTP |
| 53 | DNS |
| 80 | HTTP |
| 443 | HTTPS |
| 3306 | MySQL |
| 3389 | RDP |

---

### แท็บที่ 4: 📊 ทราฟฟิก (Traffic Monitor)

แสดงสถิติการใช้งานเครือข่ายแบบเรียลไทม์

**ข้อมูลที่แสดง:**
- 📤 **ข้อมูลส่ง**: ปริมาณข้อมูลที่ส่งออก
- 📥 **ข้อมูลรับ**: ปริมาณข้อมูลที่รับเข้า
- 📤 **แพ็กเก็ตส่ง**: จำนวนแพ็กเก็ตที่ส่ง
- 📥 **แพ็กเก็ตรับ**: จำนวนแพ็กเก็ตที่รับ
- 🔗 **การเชื่อมต่อ**: จำนวนการเชื่อมต่อ TCP ที่ใช้งานอยู่

**ตารางการเชื่อมต่อ:**
- แสดงรายละเอียดการเชื่อมต่อ TCP ทั้งหมด
- ที่อยู่ภายใน/ภายนอก
- สถานะการเชื่อมต่อ (Established, TimeWait, ฯลฯ)
- อัปเดตอัตโนมัติทุก 2 วินาที

---

### แท็บที่ 5: 🛠 เครื่องมือ (Network Tools)

รวมเครื่องมือเครือข่ายที่ใช้บ่อย 4 อย่าง:

#### 🌐 DNS Lookup
- ใส่ชื่อโดเมน แล้วกด **"ค้นหา DNS"** เพื่อดูที่อยู่ IP
- กด **"Reverse DNS"** เพื่อค้นหาชื่อโดเมนจาก IP

#### 📡 Ping
- ใส่ IP หรือโดเมน แล้วกด **"Ping"**
- จะส่ง Ping 4 ครั้ง และแสดงสถิติ

#### 🗺 Traceroute
- ใส่เป้าหมาย แล้วกด **"เริ่ม Traceroute"**
- แสดงเส้นทางที่แพ็กเก็ตเดินทาง (สูงสุด 30 hops)

#### 📋 WHOIS Lookup
- ใส่ชื่อโดเมน แล้วกด **"ค้นหา WHOIS"**
- แสดงข้อมูลการจดทะเบียนโดเมน

---

### แท็บที่ 6: 📝 บันทึก (Activity Log)

บันทึกกิจกรรมทั้งหมดของโปรแกรม

**วิธีใช้:**
- ดูบันทึกการทำงานทั้งหมด (สแกน, ดักจับ, เครื่องมือ)
- กดปุ่ม **"🗑 ล้างบันทึก"** เพื่อลบข้อมูล
- กดปุ่ม **"💾 บันทึกไฟล์"** เพื่อบันทึกเป็นไฟล์ .txt

## ความต้องการของระบบ

- **ระบบปฏิบัติการ**: Windows 10/11 (64-bit)
- **RAM**: อย่างน้อย 256 MB
- **พื้นที่ดิสก์**: ประมาณ 160 MB
- **สิทธิ์**: แนะนำให้รันด้วยสิทธิ์ Administrator เพื่อให้ทำงานได้เต็มประสิทธิภาพ

## ข้อจำกัดและหมายเหตุ

- โปรแกรมนี้เป็นเครื่องมือสำหรับ **ผู้ดูแลระบบเครือข่าย** เพื่อตรวจสอบเครือข่ายของตนเอง
- การดักจับแพ็กเก็ตใช้โหมด **จำลอง** (Simulation) เพื่อสาธิตการทำงาน
- ฟังก์ชันสแกนเครือข่ายและสแกนพอร์ตทำงานได้จริง
- **กรุณาใช้งานเฉพาะบนเครือข่ายที่คุณมีสิทธิ์เท่านั้น**
