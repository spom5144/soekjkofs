━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ขั้นตอนที่ 1: เปิดโปรแกรม
  ─────────────────────────────
  - ดับเบิลคลิก AndroidUnlockerPro.exe
  - หากมีคำเตือนจาก Windows SmartScreen ให้กด "More info" > "Run anyway"

  ขั้นตอนที่ 2: เสียบสาย USB
  ─────────────────────────────
  - เสียบสาย USB เชื่อมต่อมือถือเข้ากับคอมพิวเตอร์
  - หากมือถือถูกล็อค ให้ลองเสียบสายตอนเปิดเครื่องอยู่

  ขั้นตอนที่ 3: เชื่อมต่อ ADB
  ─────────────────────────────
  - กดปุ่ม "🔌 เชื่อมต่อ ADB" ที่ด้านบนของโปรแกรม
  - รอให้โปรแกรมตรวจพบอุปกรณ์
  - หากสำเร็จ จะแสดงสถานะ 🟢 สีเขียว พร้อมชื่อรุ่นมือถือ

  ขั้นตอนที่ 4: เลือกยี่ห้อและวิธีการ
  ─────────────────────────────────────
  - เลือกยี่ห้อมือถือจากเมนู Dropdown "ยี่ห้อมือถือ"
  - เลือกวิธีปลดล็อคจาก "วิธีปลดล็อค"
  - หากไม่แน่ใจ เลือก "อัตโนมัติ (แนะนำ)"

  ขั้นตอนที่ 5: เลือกฟังก์ชันจากเมนูด้านซ้าย
  ──────────────────────────────────────────────
  🔑 ลบรหัส Pattern     - ลบ Pattern Lock (รูปแบบปัดหน้าจอ)
  🔢 ลบรหัส PIN/Password - ลบ PIN หรือ Password
  🛡️ ปลดล็อค FRP        - ข้าม Google Account Lock หลัง Reset
  ♻️ Factory Reset       - รีเซ็ตเครื่องกลับค่าเริ่มต้น (ลบข้อมูลทั้งหมด!)
  ⚡ ปลดล็อคอัตโนมัติ    - ลองทุกวิธีอัตโนมัติ (แนะนำ!)

  ขั้นตอนที่ 6: รอผลลัพธ์
  ─────────────────────────
  - ดูผลลัพธ์ในช่อง Log ด้านล่าง
  - มือถือจะรีสตาร์ทอัตโนมัติหลังปลดล็อคสำเร็จ
  - หลังรีสตาร์ทจะสามารถเข้าใช้งานได้โดยไม่ต้องใส่รหัส

🔓 กรณีที่ 1: ลืม Pattern/PIN/Password (เปิด USB Debugging ไว้แล้ว)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ★ วิธีนี้ไม่สูญเสียข้อมูล! ★

  1. เสียบ USB > กด "เชื่อมต่อ ADB"
  2. กด "🔑 ลบรหัส Pattern" หรือ "🔢 ลบรหัส PIN/Password"
  3. รอจนเสร็จ > มือถือรีสตาร์ท > เข้าใช้งานได้!

🔓 กรณีที่ 2: ลืม Pattern/PIN/Password (ไม่ได้เปิด USB Debugging)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ★ ต้อง Factory Reset (สูญเสียข้อมูลทั้งหมด!) ★

  วิธี A: ผ่าน Google Find My Device (ไม่ต้องเสียบ USB)
  1. เปิดเว็บ google.com/android/find บนคอมพิวเตอร์
  2. ล็อกอิน Google Account ที่ใช้ในมือถือ
  3. กด "ลบข้อมูลอุปกรณ์" (Erase Device)

  วิธี B: ผ่าน Recovery Mode
  1. ปิดเครื่องมือถือ
  2. เข้า Recovery Mode (ดูปุ่มกดของแต่ละยี่ห้อด้านล่าง)
  3. เลือก "Wipe data/factory reset"
  4. ยืนยัน > รอจนเสร็จ > เลือก "Reboot system now"

📱 วิธีเปิด USB Debugging (สำหรับอนาคต)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. ไปที่ ตั้งค่า (Settings)
  2. เลื่อนลงไปที่ เกี่ยวกับโทรศัพท์ (About Phone)
  3. กด "หมายเลข Build" (Build Number) ติดต่อกัน 7 ครั้ง
  4. จะปรากฏข้อความ "คุณเป็นนักพัฒนาแล้ว!"
  5. กลับไปที่ ตั้งค่า > ตัวเลือกนักพัฒนา (Developer Options)
  6. เปิด USB Debugging (การดีบัก USB)

  ★ แนะนำให้เปิดไว้เสมอ เผื่อลืมรหัสในอนาคต!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ❌ ไม่พบอุปกรณ์
     → ลองเปลี่ยนสาย USB (ใช้สายที่รองรับ Data)
     → ลองเปลี่ยนพอร์ต USB
     → ติดตั้งไดรเวอร์ USB ของยี่ห้อนั้นๆ
     → กด "ตัดการเชื่อมต่อ" แล้ว "เชื่อมต่อ ADB" ใหม่

  ❌ ADB Unauthorized
     → ดูหน้าจอมือถือ กด "อนุญาต" (Allow)
     → หากมือถือล็อค ต้องใช้ Recovery Mode แทน

  ❌ ปลดล็อคแล้วยังมีรหัส
     → ลองรีบูตเครื่องอีกครั้ง
     → ลองกด "ปลดล็อคอัตโนมัติ"
     → ใช้ Factory Reset เป็นทางเลือกสุดท้าย

  ❌ FRP Lock หลัง Factory Reset
     → จำ Google Account เดิมได้: ล็อกอินปกติ
     → จำไม่ได้: ใช้ฟีเจอร์ "ปลดล็อค FRP" ในโปรแกรม
